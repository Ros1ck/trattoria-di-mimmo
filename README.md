# Trattoria Pizzeria Da Mimmo — sito multipagina

Sito statico pronto per GitHub Pages.

## Pagine
- `index.html` — Home
- `menu.html` — Menu
- `chi-siamo.html` — La trattoria
- `galleria.html` — Galleria
- `prenotazioni.html` — Prenotazioni e contatti

## Pubblicazione su GitHub
1. Scarica ed estrai il file ZIP.
2. Nel repository GitHub vuoto premi **uploading an existing file**.
3. Trascina **tutti i file e le cartelle estratti**, non il file ZIP.
4. Premi **Commit changes**.
5. Vai in **Settings → Pages**.
6. In **Build and deployment**, scegli **Deploy from a branch**.
7. Seleziona `main` e `/ (root)`, poi salva.

## Da verificare prima dell’uso reale
- indirizzo e orari;
- menu, prezzi, ingredienti e allergeni;
- fotografie reali del locale;
- conferma che il numero mobile possa ricevere messaggi WhatsApp.
